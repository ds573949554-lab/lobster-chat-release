## 🚀 快速開始

### 1. 獲取 API Key

選擇你想用的 AI 模型，訪問對應網站：

| 模型 | 獲取 API Key 網址 |
|------|------------------|
| Gemini 3 Pro | https://aistudio.google.com/app/apikey |
| Claude Opus 4.5 | https://console.anthropic.com/ |
| GPT-5 | https://platform.openai.com/api-keys |
| Kimi K2.5 | https://platform.moonshot.cn/ |
| GLM-4.7 | https://open.bigmodel.cn/ |
| Qwen3-Max | https://dashscope.aliyun.com/ |
| DeepSeek V3.2 | https://platform.deepseek.com/ |

### 2. 配置 API Key

- 打開 App → 設置
- 點擊你想用嘅 AI 模型
- 輸入 API Key
- 點「選擇」啟用該模型

### 3. 開始對話

- 回到對話頁面
- 輸入問題，AI 會即時回應
