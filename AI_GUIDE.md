## 使用方法

### 1. 配置 AI API Key
- 選擇你想用的 AI 模型
- 訪問對應網站註冊並獲取 API Key
- 打開 App → 設置 → 點擊「添加」輸入 API Key
- 點擊「選擇」啟用該模型

### 2. 開始對話
- 回到對話頁面
- 輸入問題，AI 會即時回應

### 3. 🔊 語音朗讀
- **自動朗讀**：開啟「自動朗讀」開關，AI 回應會自動語音播放
- **手動朗讀**：點擊每條 AI 消息下方的 🔊 朗讀按鈕
- **停止播放**：點擊 ⏹️ 停止按鈕

## 安全提示

- ✅ API Key 只存儲在本地設備
- ✅ 不會上傳到任何服務器
- ⚠️ 請勿分享你的 API Key

## 常見問題

**Q: 可以同時配置多個 AI 嗎？**
A: 可以！配置多個後可隨時切換。

**Q: 需要付費嗎？**
A: 大部分 AI 都有免費額度，具體請查看各平台定價。

**Q: 為什麼對話失敗？**
A: 請檢查：1) API Key 是否正確 2) 網絡連接 3) 該模型是否有餘額
